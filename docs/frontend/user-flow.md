# 前端用户流程

## 完整演示流程

1. 访问首页。
2. 使用 `demo_student / 123456` 登录。
3. 进入 Dashboard 查看学习闭环概览。
4. 进入学习空间页确认或创建学习空间。
5. 进入模型配置页创建 Mock 模型并测试连接。
6. 进入智能对话页发送学习问题。
7. 进入 Agent 工作台生成学习资源。
8. 进入知识库页创建资料元数据、索引并检索。
9. 进入资源生成中心生成知识图谱。
10. 进入知识图谱页查看 Mermaid 渲染结果。
11. 进入学习路径页生成路径并标记任务完成。
12. 进入测验页生成测验、答题并查看掌握度。
13. 进入报告页生成学习报告并导出 Markdown。

## 服务边界

前端只调用 Java 后端 `/api/**`。Python AI 服务由 Java 后端调用，前端不直接访问 Python 服务。
