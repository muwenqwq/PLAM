# 前端页面说明

## 页面结构

前端采用 `BasicLayout` 作为登录后统一布局，包含左侧菜单、顶部用户信息、退出按钮和主内容区。首页、登录和注册不使用业务布局。

## 核心页面

| 路径 | 页面 | 说明 |
|---|---|---|
| `/dashboard` | 学习驾驶舱 | 汇总资源、测验、路径、掌握度并展示图表 |
| `/spaces` | 学习空间 | 创建、编辑、删除、设置默认学习空间 |
| `/profile` | 用户画像 | 维护学习目标、薄弱点、偏好和输出风格 |
| `/models` | 模型配置 | 创建 Mock 模型、测试连接、设置默认模型 |
| `/chat` | 智能对话 | 创建会话并发送学习问题 |
| `/agents` | Agent 工作台 | 创建多智能体任务并展示执行步骤 |
| `/knowledge` | 知识库 | 创建资料元数据、索引、检索和问答 |
| `/resources` | 资源生成 | 生成讲义、提纲、习题集、知识图谱和 PPT 大纲 |
| `/graph` | 知识图谱 | 渲染 Mermaid 知识图谱并复制源码 |
| `/paths` | 学习路径 | 生成路径、今日任务、标记完成和调整路径 |
| `/quiz` | 测验评估 | 生成测验、答题、评分和掌握度展示 |
| `/reports` | 学习报告 | 汇总概览、生成报告、导出 Markdown |

## 权限规则

除首页、登录、注册外，其余页面均要求登录。登录状态由 Pinia 和 localStorage 保存，Axios 自动携带 `Authorization: Bearer <token>`。
