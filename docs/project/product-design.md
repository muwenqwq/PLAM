# 产品设计说明

## 1. 产品定位

智学工坊 EduAgent Studio 是一个面向学习资源生成和学习过程管理的智能学习平台。产品目标不是单一聊天机器人，而是一个具备学习空间、用户画像、知识库、多智能体任务、测验和学习报告闭环的完整软件系统。

## 2. 核心用户场景

### 2.1 学生复习场景

学生创建“数据库系统期末复习”学习空间，填写基础水平和考试时间，上传课程资料，通过智能对话提出“帮我复习范式、SQL 查询和索引”。系统识别学习意图，调用多智能体生成复习计划、知识图谱、讲义、习题和测验，最后根据测验结果生成学习报告。

### 2.2 教师备课场景

教师创建课程空间，配置课程目标和难度，上传参考材料，生成课堂讲义、案例任务、阶段测验和答疑提纲。教师可以查看生成资源质量评分和修改建议。

### 2.3 自学项目场景

自学用户创建“Java 后端项目实战”学习空间，要求系统生成阶段性学习路径、实践任务和测验题。系统根据完成情况调整学习路径，并输出学习报告。

## 3. 页面设计

1. 登录页：用户名、密码、登录按钮、演示账号提示。
2. 注册页：用户名、昵称、邮箱、密码和确认密码。
3. Dashboard：学习空间数量、资源数量、测验成绩、近期任务和学习趋势。
4. 学习空间页：空间列表、创建空间、查看详情和统计摘要。
5. 用户画像页：学习目标、基础水平、兴趣标签、学习时间和资源偏好。
6. AI 模型配置页：模型类型、Base URL、模型名、API Key 脱敏展示、默认模型、连接测试。
7. 智能对话页：历史会话、消息列表、Markdown 回复和转为资源任务。
8. 多智能体工作台：任务输入、智能体步骤时间线、输出摘要和最终资源。
9. 知识库页：文件上传、文件列表、解析状态、Chunk 数量和检索入口。
10. 学习资源生成页：资源类型、学科、知识点、难度、是否启用知识库、生成结果。
11. 知识图谱页：Mermaid 图谱渲染、源码复制和重新生成。
12. 学习路径页：阶段任务、完成状态、今日任务和进度条。
13. 在线测验页：题目展示、答案提交、评分结果和解析。
14. 学习报告页：统计卡片、ECharts 图表、文字建议和导出。
15. 帮助页：使用说明、Mock 模式说明和演示流程。

## 4. 产品闭环

产品闭环为：创建学习空间 -> 配置画像和模型 -> 上传资料 -> 对话识别意图 -> 多智能体生成资源 -> 资源保存和导出 -> 在线测验 -> 掌握度更新 -> 学习报告。

该闭环保证系统不是孤立的 AI Demo，而是具备数据沉淀、业务流程和可视化反馈的软件系统。
