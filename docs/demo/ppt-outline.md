# 答辩 PPT 大纲

## 1. 项目标题与团队信息

标题：智学工坊 EduAgent Studio。说明 LearnAgent-A3 与 EduAgent Studio 为同一项目的命名过渡。

## 2. 赛题背景与痛点

学生学习资源分散、个性化路径不足、AI 输出难以沉淀、复习效果缺少反馈。

## 3. 系统目标与应用场景

面向课程复习、备考、教师备课和软件实践，形成“目标输入-资源生成-路径规划-测验反馈-报告复盘”闭环。

## 4. 总体架构

展示 Vue 前端、Java 后端、MySQL、Redis、Python AI 服务和 Mock/真实模型适配。

## 5. 数据库设计

展示用户、学习空间、模型配置、智能体任务、资源、知识库、学习路径、测验、报告等核心表关系。

## 6. Java 后端模块

说明认证安全、统一响应、异常处理、MyBatis-Plus、模块化 Controller-Service-Mapper 结构。

## 7. Python AI 微服务与多智能体

说明 Planner、Knowledge、Exercise、Review 等智能体角色，以及 Mock Provider 的演示保障。

## 8. 前端功能展示

展示仪表盘、模型配置、智能对话、智能体工作台、知识库、资源生成、学习路径、测验、报告页面。

## 9. 核心演示流程

使用 `demo_student / 123456` 完成登录、生成资源、学习路径、测验和报告。

## 10. 创新点

个性化学习画像、多智能体协作、结构化资源沉淀、Mock AI 保底演示、Java+MySQL 规范化工程。

## 11. 测试与部署

展示单元测试、前端构建、Smoke Test、Docker Compose 一键启动。

## 12. 总结与展望

总结已实现闭环，展望真实模型接入、向量数据库、文件解析、异步任务和权限细化。
