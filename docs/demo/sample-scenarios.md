# 演示样例场景

## 场景一：学生期末复习

用户：`demo_student`

目标：两周内完成数据库系统期末复习。

演示动作：

1. 登录系统。
2. 进入“数据库系统期末复习”学习空间。
3. 使用 Mock 模型测试连接。
4. 与 AI 对话询问第二范式、SQL 查询优化和索引设计。
5. 创建智能体任务生成复习计划。
6. 生成知识图谱和 7 天学习路径。
7. 生成测验并提交答案。
8. 查看学习报告。

## 场景二：教师备课

用户：`demo_teacher`

目标：为 Java 后端与数据库课程准备讲义、案例和测验。

演示动作：

1. 查看教师学习空间。
2. 配置教师 Mock 模型。
3. 生成课程讲义和案例任务。
4. 生成课堂测验。
5. 查看资源质量评分和报告。

## 场景三：管理员验收

用户：`demo_admin`

目标：展示系统工程完整性。

演示动作：

1. 查看接口文档。
2. 说明数据库表设计和逻辑外键策略。
3. 执行 Smoke Test。
4. 展示 Docker Compose 配置和启动脚本。
