# 演示场景样例

## 场景一：数据库期末复习

用户使用 `demo_student / 123456` 登录后，以“数据库”为学习空间，生成范式、SQL 查询和索引相关资料。系统通过 Mock AI 生成资源、知识图谱、学习路径和测验，并最终生成学习报告。

## 场景二：Java 后端复习

用户创建 Java 后端学习目标，使用智能对话询问 Spring Boot、MyBatis-Plus 和 JWT。随后通过 Agent 工作台生成复习计划，并在学习路径页跟踪任务。

## 场景三：算法知识点讲解

用户在智能对话中输入“A* 搜索算法”，系统返回 Markdown 讲解。用户可进一步生成知识图谱和习题集，用于答辩展示。
