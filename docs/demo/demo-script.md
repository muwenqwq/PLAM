# 7 分钟答辩演示脚本

## 演示准备

- 前端地址：`http://127.0.0.1:5173`
- 后端健康检查：`http://127.0.0.1:8080/api/health`
- AI 服务健康检查：`http://127.0.0.1:8000/ai/health`
- 演示账号：`demo_student / 123456`
- 默认模式：Mock AI，无需真实 API Key。

## 0:00-0:40 项目开场

介绍项目名称“智学工坊 EduAgent Studio”，说明 LearnAgent-A3 是前期赛题文档名称。强调系统目标是把 AI 生成能力转化为可管理的学习空间、学习资源、测验和报告闭环。

## 0:40-1:20 登录与仪表盘

使用 `demo_student / 123456` 登录，进入 Dashboard。展示学习空间数量、资源数量、测验记录、学习进度和最近活动，说明前端通过 JWT 调用 Java 后端。

## 1:20-2:00 学习空间与模型配置

进入学习空间页面，展示“数据库系统期末复习”空间。进入模型配置页面，展示 Mock 模型配置，点击测试连接，说明系统不保存明文 API Key，Mock 模式保证现场稳定演示。

## 2:00-2:50 智能对话

进入 AI 对话页面，发送“请帮我解释第二范式并给出复习建议”。展示后端调用 Python AI 服务返回结构化回答，并沉淀会话记录。

## 2:50-3:40 智能体工作台

进入智能体工作台，创建资源生成任务。展示 PlannerAgent、KnowledgeAgent、ExerciseAgent、ReviewAgent 等步骤，说明多智能体把任务拆解为规划、知识梳理、练习生成和质量复盘。

## 3:40-4:30 知识库与资源生成

进入知识库页面，展示文件元数据、检索和问答。进入资源生成页面，生成复习计划或知识图谱，说明生成结果保存到 `generated_resource` 表，可继续用于学习路径和报告。

## 4:30-5:20 知识图谱与学习路径

进入知识图谱页面，展示 Mermaid 图谱。进入学习路径页面，生成 7 天复习路径，说明系统会将学习目标拆成可跟踪任务。

## 5:20-6:10 测验与报告

进入测验页面生成测验并提交答案，展示得分和反馈。进入报告页面查看学习概览和建议，说明系统形成“生成-练习-反馈-复盘”的闭环。

## 6:10-7:00 技术总结

总结技术架构：Vue 前端、Spring Boot 后端、MySQL 数据库、Python AI 微服务、Docker Compose 部署。强调数据库规范化设计、逻辑外键策略、多智能体 Mock 演示和后续真实模型扩展能力。
