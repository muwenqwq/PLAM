# 7 分钟答辩演示脚本

## 0:00 - 0:40 项目定位

说明本项目是面向软件杯 A3 赛题的“个性化资源生成与学习多智能体系统”，采用 Java 主后端、MySQL 主数据库、Python AI 微服务和 Vue 前端。

## 0:40 - 1:20 登录与 Dashboard

使用 `demo_student / 123456` 登录，进入 Dashboard，展示学习空间、资源、测验、路径、平均分和掌握度。

## 1:20 - 2:00 模型配置

进入模型配置页，创建 Mock 模型并测试连接，强调不保存明文 API Key，只展示脱敏字段。

## 2:00 - 2:50 智能对话

进入智能对话页，发送“我两天后要考数据库，帮我复习范式、SQL 查询和索引。”，展示 Markdown 回复。

## 2:50 - 3:40 Agent 工作台

创建多智能体任务，展示 Planner、Knowledge、Exercise、Review 的执行时间线和最终资源。

## 3:40 - 4:40 知识库与资源生成

创建知识库资料元数据，调用索引，执行检索和问答。随后在资源生成中心生成知识图谱。

## 4:40 - 5:30 学习路径

生成一周学习路径，展示路径节点和今日任务，并标记一个节点完成。

## 5:30 - 6:20 测验与掌握度

生成测验、提交答案，展示得分、正确率、薄弱知识点和掌握度记录。

## 6:20 - 7:00 学习报告

进入报告页，生成学习报告，展示 ECharts 图表和 AI 学习建议，导出 Markdown。
