# 学生端参考 UI 重构设计

## 目标

在不改变现有业务接口、数据结构和核心路由的前提下，把学生端整体视觉重构为参考目录中的学习工作台风格。覆盖公开首页、登录、注册和所有登录后页面。

## 视觉系统

- 默认主题为白色，使用浅灰背景、半透明白色面板、紫色主色和蓝色辅助色。
- 夜间主题延续参考原型的深色玻璃界面，使用近黑背景、低透明度面板和浅紫色强调。
- 组件采用 12px 至 16px 的克制圆角、轻边框和低强度阴影，不使用装饰性光球。
- 导航图标统一使用 Element Plus 图标，禁止使用 Emoji 作为结构图标。
- 所有颜色通过语义 CSS 变量提供，页面不得依赖只适合单一主题的硬编码白色背景。

## 应用外壳

- 桌面端保留左侧导航和顶部工作区，侧栏支持折叠。
- 移动端侧栏改为抽屉，并提供遮罩和明确关闭路径。
- 顶部显示当前页面标题、说明、主题切换、用户头像和退出入口。
- 左侧仍只显示八个学生核心入口，隐藏的开发功能不重新暴露。

## 页面

- 首页、登录和注册采用同一品牌与主题系统。
- 学习首页使用欢迎区、快捷学习入口、统计概览和最近内容。
- 知识库、资源生成、我的资源、测验和报告保留真实接口与交互，只重排视觉层级。
- 空状态、加载状态、危险操作、表格、弹窗和表单在两种主题下保持可读。

## 主题行为

- 首次访问默认白色主题。
- 用户切换后保存到 `localStorage` 的 `eduagent_theme`。
- 页面加载前尽早恢复主题，减少闪烁。
- 切换按钮在公开页、登录、注册和学生工作台中均可访问。

## 验证

- 运行主题契约测试。
- 运行 `vue-tsc` 与 Vite 生产构建。
- 使用本机 Chrome 对主要路由进行桌面和移动视口截图。
- 检查控制台错误、水平溢出、主题切换持久化和导航可达性。
