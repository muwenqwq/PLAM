# 智能对话模块说明

## 1. 模块定位

智能对话模块负责保存学习会话、用户消息和 AI 回复。Java 后端负责会话与消息持久化，Python AI 服务只返回结构化回复内容。

## 2. 当前流程

1. 用户创建会话。
2. 用户发送消息。
3. Java 保存用户消息。
4. Java 选择指定模型或默认模型；没有配置时使用 Mock。
5. Java 调用 Python `/ai/chat`。
6. Python Mock 返回 Markdown 回复和结构化 JSON。
7. Java 保存 AI 回复。
8. 接口返回用户消息和 AI 消息。

## 3. 数据隔离

所有会话和消息都包含 `user_id`。查询、发送消息和读取消息时，Service 层统一校验当前登录用户只能访问自己的会话。
