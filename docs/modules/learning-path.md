# 学习路径模块说明

## 模块定位

学习路径模块基于用户画像、学习目标、学习空间和知识点生成阶段化任务，并支持任务状态更新、今日任务查询和 Mock 路径调整。

## 数据表

- `learning_path`：路径主表，保存标题、目标、学科、计划 JSON、进度和日期。
- `learning_path_item`：路径任务项，保存任务顺序、知识点、预计时长、截止日期和完成状态。

## 状态规则

路径任务状态包括 `todo`、`doing`、`done`、`skipped`。当任务被更新为 `done` 时写入 `completed_at`，并重新计算路径 `progress_rate`。
