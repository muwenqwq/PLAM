# 学习空间模块说明

## 1. 模块定位

学习空间是 EduAgent Studio / LearnAgent-A3 的核心业务容器，用于承载某一门课程、某个考试或某个学习目标下的对话、知识库、智能体任务、生成资源、学习路径、测验和报告。

当前阶段只实现学习空间自身的基础 CRUD、默认空间管理和摘要查询，不实现 AI 对话、资源生成和任务执行。

## 2. 功能范围

| 功能 | 说明 |
|---|---|
| 创建学习空间 | 当前用户创建学习空间，第一个空间自动设为默认空间 |
| 分页查询 | 按当前用户查询，支持页码、每页条数、关键词、学科和状态过滤 |
| 详情查询 | 只能查询当前用户自己的学习空间 |
| 更新学习空间 | 支持名称、学科、描述、封面、可见性和状态维护 |
| 逻辑删除 | 使用 MyBatis-Plus 逻辑删除，不物理删除数据 |
| 默认空间 | 设置某空间为默认时取消当前用户其他默认空间 |
| 摘要查询 | 当前阶段返回缓存统计字段，后续对接真实任务和资源统计 |

## 3. 数据隔离策略

所有学习空间查询和写入都通过 `LoginUserHolder.requireCurrentUserId()` 获取当前用户 ID，并在 Service 层统一附加 `user_id` 条件。不存在或不属于当前用户的学习空间统一返回“学习空间不存在或无权访问”，避免暴露资源归属信息。

## 4. 默认学习空间逻辑

`learning_space.is_default` 用于标记默认学习空间。默认约束由 Java Service 层维护：

1. 创建当前用户第一个学习空间时自动设为默认。
2. 设置某个空间为默认时，先取消当前用户其他空间的默认状态，再设置目标空间。
3. 删除默认空间时，如果用户仍有其他空间，则选择最近更新的一条作为新的默认空间；如果没有剩余空间，则默认空间为空。

## 5. 后续衔接

后续 AI 对话、知识库文件、智能体任务、生成资源、学习路径、测验和报告均会通过 `space_id` 归属到学习空间。学习空间详情页可以聚合展示这些模块的运行状态和学习成果。
