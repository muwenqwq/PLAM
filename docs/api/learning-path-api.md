# 学习路径 API

| 方法 | 路径 | 说明 |
|---|---|---|
| POST | `/api/learning-paths/generate` | 生成学习路径 |
| GET | `/api/learning-paths` | 分页查询路径 |
| GET | `/api/learning-paths/{id}` | 查询路径详情 |
| PUT | `/api/learning-path-items/{id}/status` | 更新路径任务状态 |
| GET | `/api/learning-paths/today` | 查询今日任务 |
| POST | `/api/learning-paths/{id}/adjust` | Mock 调整路径 |
