智学工坊 EduAgent Studio - 第三方组件许可说明
===========================================

本运行包为方便评审和演示，随包提供以下第三方运行组件：

1. MySQL Community Server 8.4
   许可证文件位于 runtime\mysql\LICENSE。
   MySQL 仅在本机 127.0.0.1:33306 为本作品提供独立数据服务。

2. Java 21 运行时
   运行时版权与许可证文件位于 runtime\jre\legal 及其随附文件中。

3. AI 服务可执行程序
   由项目 Python 源码及其依赖打包生成。对应源码依赖版本见源码包 ai-service\requirements.txt。

4. 前端与后端依赖
   前端依赖声明见源码包 frontend\package.json；后端依赖声明见源码包 backend\pom.xml。

本文件仅用于说明随包组件位置，不替代各组件原始许可证文本。
