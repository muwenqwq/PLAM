智学工坊 EduAgent Studio - Windows 作品安装/可执行包
======================================================

一、系统要求
1. Windows 10/11 64 位。
2. 至少 2 GB 可用磁盘空间，建议 4 GB 以上可用内存。
3. 本包已包含 MySQL 8、Java 21、前端页面、后端程序和 AI 服务，不需要安装数据库、Node、Maven 或 Python。

二、首次使用
1. 完整解压整个压缩包，不能只运行压缩包内的单个文件。
2. 直接双击“启动智学工坊.bat”。首次运行会自动初始化作品内置数据库。
3. 浏览器会自动打开系统；也可以手动访问 http://127.0.0.1:18080。
4. 也可以先双击“安装并初始化.bat”，再启动系统。
5. 首次初始化通常需要 20～60 秒。安装脚本已适配包含中文和空格的本地路径，请勿放在只读目录或网络共享目录中运行。

三、演示账号
用户名：demo_student
密码：123456

四、停止服务
双击“停止智学工坊.bat”。用户上传资料、数据库和学习数据均保存在本作品的 data 目录中，不会写入电脑上已有的 MySQL 数据库。

五、模型配置
系统内置稳定演示能力，可以直接体验完整学习流程。需要接入真实大模型时，在系统“AI 模型”页面添加兼容 OpenAI API 的模型配置，并填写服务地址、模型名和 API Key。提交包不包含任何真实密钥。

六、常见问题
1. 提示端口占用：关闭占用 33306、18000 或 18080 端口的程序后重试。
2. 启动失败：查看 logs 目录中的 mysql、ai 和 backend 日志。
3. 需要恢复初始演示状态时，双击“重置演示数据.bat”，并按提示输入 RESET。
4. 请勿单独移动 runtime、app、database、config 或 scripts 目录。

七、第三方运行时
本包包含 MySQL Community Server 8.4 与 Java 21 运行时。相关许可证文件随 runtime 目录一并提供。

八、端口与数据目录
1. MySQL：127.0.0.1:33306，仅供本作品内部使用。
2. AI 服务：127.0.0.1:18000。
3. Web 系统：127.0.0.1:18080。
4. 用户资料、学习记录和数据库文件均保存在本包 data 目录，停止服务不会删除数据。
